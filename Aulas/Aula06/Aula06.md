# **Coleções**
são estruturas de dados abstratas
    - List -> vector, ArrayList, LinkedList
      - Listas para armazenamento de dados, remoção, busca e relatórios
    - Stack (pilha) - LIFO -> last in, first out - o último que entra é o primeiro a sair
      - TOPO - peek
      - Há somente duas operações clássicas: REMOVER do topo e INSERIR no topo
        - push() - INSERIR
        - pop() - remover - NÃO SE ESCOLHE QUEM SERÁ REMOVIDO... A REMOÇÃO ACONTECE NO TOPO
      - OUTROS métodos básicos
        - peek() - mostra o topo
        - size() - mostra o tamanho
        - isEmpty() - verifica se está vazia
        - contains() - procura por um elemento na pilha
    - Não se **exibe** pilha

    - Fila (Queue) - FIFO - First in, First out - o primeiro que chega é o primeiro a sair
    - Dois Ponteiros: Um para head (início); outro para a Cauda/fim (inserir)
    - Há somente duas operações clássicas: Remover head e inserir na Cauda
      - add()
      - remove() 
    - FILA é uma forma de gerenciar prioridade, o primeiro que chega é o primeiro a ser atendido
    - Toda fila é uma lista
